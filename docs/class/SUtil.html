Class Method Details

. (Null) reportChange(callback, context, args...)

Function that calls the callback with the given args and context

Examples:

Calling func with ctx as this and no arguments: 

SUtil.reportChange(func, ctx)

Calling the func with ctx as this and two arguments: 

SUtil.reportChange(func, ctx, 1, 2)

Parameters:

  • callback ( Function ) — function that will be called back
  • context ( Object ) — this context that will be applied to callback
  • args ( Array ) — (Optional )A splat with the arguments that will be passed along

Returns:

  • ( Null ) — it will return null if can't call the callback, undefined otherwise

Since:

  • 0.1.0

. (String) getType(target)

Reliably gets the JS type of the given variable

Examples:

Checking a string will return [object String] 

SUtil.getType('hello')

Checking a number will return [object Number] 

SUtil.getType(1)

Parameters:

  • target ( Object ) — variable to be type tested

Returns:

  • ( String ) — The type of the given variable

Since:

  • 0.1.0

. (Boolean) isValidEventName(name)

Checks if the suplied variable can be used as event name for ScrollProxy @param [String name Variable to be tested as valid event name

Examples:

Empty strings will return false 

SUtil.isValidEventName('')

Non-string object will return false 

SUtil.isValidEventName(1)

String with one or more chars will return true 

SUtil.isValidEventName('myEvent')

Returns:

  • ( Boolean ) — true if valid name and false if not

Since:

  • 0.1.0

. (Boolean) isPositiveNumber(number)

Checks if the suplied variable is a positive number, it also considers 0 a positive number @param [String name Variable to be tested as positive number
