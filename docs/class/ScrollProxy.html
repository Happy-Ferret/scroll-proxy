Class: ScrollProxy

Defined in: src/ScrollProxy.coffee

Overview

Main class, containing the scrolling related events and API

Options Hash: (_targetList):

  • target ( HTMLElement ) — The anchor of the ScrollProxy instance
  • proxy ( ScrollProxy ) — The instance of ScrollProxy

Options Hash: (_events):

  • type ( String ) — The event name of the event
  • fn ( Function ) — The function that will handle the callback
  • timeout ( Number ) — The timeout id of the throttled fn

# (Function) _throttleScroll(func, delay) Private

Function that performs the scroll throttling to achieve better performance

Examples:

One must pass a function to be throttled and a delay: 

s = new ScrollProxy()
func = -> this
newFunc = s._throttleScroll(func, 250)
newFunc() # func will be called after 250ms, the scrollData will be passed
newFunc() # func will not be called, because other call is waiting

Parameters:

  • The ( Function ) — function to throttle
  • delay ( Number ) — The time function should wait to fire

Returns:

  • ( Function ) — The new throttled function

  • 0.1.0

# (ScrollProxy) on(evt, func, option)

Register events in ScrollProxy

Examples:

Registering a new scroll event on body: 

s = new ScrollProxy()
s.on('scroll', -> 'Awesome scrolling!')

Parameters:

  • evt ( String ) — The event name to Register
  • func ( Function ) — The function to callback once event is fired
  • option ( Object ) — A wildcard argument that is passed to the event

Events:

scroll

The scroll event that is fired whenever one scrolls

offsetX

The offset event on the X axis that fires once one scrolls past the given horizontal offset

offsetY

The offset event on the Y axis that fires once one scrolls past the given vertical offset

top

The event that fires once one reaches the top of the scroll area or the given offset top

bottom

The event that fires once one reaches the bottom of the scroll area or the given offset bottom

left

The event that fires once one reaches the left bound of the scroll area of the given offset left

right

The event that fires once one reaches the right bound of the scroll area of the given offset right

visible

The event that fires once the given element is visible on the scroll area

invisible

The event that fires once the given element is not visible on the scroll area

Returns:

  • ( ScrollProxy ) — This instance

Since:

  • 0.1.0

Register this target on ScrollProxy's _targetList
